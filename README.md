# Java OOPs Mini Projects 🚀
A collection of console-based mini projects built using Java to practice Object-Oriented Programming (OOPs) concepts.

## 1. ATM Simulation System 🏧
A simple console application that mimics real-life ATM functionalities.

### 🛠 Concepts Covered:
- **Encapsulation:** Using private variables for PIN and Balance.
- **Abstraction:** Hiding complex logic behind simple methods.
- **Control Flow:** Using `switch-case` and `do-while` loops.

### ✨ Features:
- PIN Verification
- Balance Inquiry
- Cash Deposit & Withdrawal
- **Secure PIN Change** (with old PIN validation)

### 🚀 How to Run:
1. Navigate to `ATM-System` folder.
2. Compile: `javac ATMApp.java`
3. Run: `java ATMApp`
